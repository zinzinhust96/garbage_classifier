# Garbage classifier

## How to install

### Install miniconda
### Create virtual environment
```bash
$ conda create -n vision python=3.11
```

### Activate virtual environment
```bash
$ conda activate vision
```

### Install dependencies
```bash
$ pip install -r requirements.txt
```

## How to use

### Activate virtual environment
```bash
$ conda activate vision
```

### Demo
```bash
$ python capture.py
```
